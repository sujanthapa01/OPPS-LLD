# Offline Wallet
A simple TypeScript wallet system for transferring money between users.

Structure
```bash offline_wallet/
├── functions/
│   └── getCurrentDateTime.ts
├── models/
│   ├── User.ts
│   └── wallet.ts
└── services/
    └── transfer_service.ts

root ── types/
    └── offline_wallet_types/
```
